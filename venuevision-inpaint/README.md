# VenueVision Inpaint

Click on objects in wall images to remove them. Uses **SAM2** + **LaMa** inpainting.

## API for VenueVision

Set in your `.env`:
```
INPAINT_SPACE_URL=TahaMunshi03/venuevision-inpaint
```
