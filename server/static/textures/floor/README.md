# Floor Textures

Place your floor texture images here. Supported formats: JPG, PNG.

**Expected filenames** (used by the Floor Planner):
- `interior_tiles_diff_4k.jpg`
- `rubber_tiles_diff_4k.jpg`
- `wood_floor_worn_diff_4k.jpg`
- `wood_table_001_diff_4k.jpg`

You can add more files; they will appear in the Floor Planner if you add them to the dropdown options in the code.
