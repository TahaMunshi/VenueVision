# Ceiling Textures

Place your ceiling texture images here. Supported formats: JPG, PNG.

**Expected filenames** (used by the Floor Planner):
- `plank_flooring_04_diff_4k.jpg`
- `plastered_wall_02_diff_4k.jpg`

You can add more files; they will appear in the Floor Planner if you add them to the dropdown options in the code.
